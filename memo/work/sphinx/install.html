Spinx Project の作成¶

sphinx-quickstart

  1. 拡張テーマの適用のため、conf.pyに以下を追記・書き換えする。

import sphinx_rtd_theme

html_theme = 'sphinx_rtd_theme'
html_theme_path = [sphinx_rtd_theme.get_html_theme_path()]

  1. MarkDown形式のファイルを利用可能にするため、conf.pyに以下を追記・書き換えする。

source_suffix = {
    '.rst': 'restructuredtext',
    '.md': 'markdown',
}

  1. Githubページへの適用のため、ビルドのディレクトリ構成を変更する。 Windowsを利用しているため、make.batに以下の設定を追記・書き換え

set BUILDDIR=.
set ALLSPHINXOPTS=-d %BUILDDIR%/doctrees %SPHINXOPTS% source
set I18NSPHINXOPTS=%SPHINXOPTS% source


if "%1" == "html" (
	%SPHINXBUILD% -b html %ALLSPHINXOPTS% %BUILDDIR%
	if errorlevel 1 exit /b 1
	echo.
	echo.Build finished. The HTML pages are in %BUILDDIR%.
	goto end
)

GitHub Pageの作成¶

  1. Github上でレポジトリ(doc)を作成した後、masterと gh-pagesという名前のブランチを作成する

git clone https://github.com/debugrecord/docs.git
cd doc
git branch master
git branch gh-pages

  1. 作成したgh-pagesブランチに切り替え、sphinxsプロジェクトコピーし、htmlでビルドする。(index.htmlがルートに来るようにする。)

git checkout gh-pages
cp somepath/test_sphinx .
make html

  1. ビルドして出来たHTMLをプッシュし、http://xxxxxx.github.io/projectへアクセスする。

git push origin gh-pages
